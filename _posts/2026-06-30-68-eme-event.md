Nous avons le plaisir de recevoir [Antoine DESSAIGNE](https://www.linkedin.com/in/antoine-dessaigne-3618aa37) principal software architect chez Axway.

Voici ce qu'il nous propose :

> Tout commence avec une mise à jour de Java 17 vers Java 21.0.1 qui fait planter un de nos benchmarks. Notre unique symptôme: une variable locale non-null envoie une NullPointerException.
>
> Je vous emmène dans mon exploration de la JVM, de la découverte de son fonctionnement jusqu'à la root-cause de ce bug.
>
> À l'issue de cette présentation, vous comprendrez que vous auriez pu le faire, que c'était à votre portée. Sans le savoir, vous êtes déjà capable de compiler votre propre JDK pour diagnostiquer et corriger des bugs. Qui sait, vous deviendrez peut-être contributeur Java ?

La présentation sera suivie d’une discussion libre autour des thèmes abordés par Antoine.

Le meetup se passe dans les locaux de [SII](https://sii-group.com/fr-FR/sii-atlantique){:target="_blank"} Tours.

Les inscriptions se passent sur [eventbrite](https://www.eventbrite.fr/e/billets-toi-aussi-ose-participer-a-lun-des-projets-open-source-les-plus-utilise-1991965212573){:target="_blank"}.
